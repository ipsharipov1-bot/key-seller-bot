# 🚀 Быстрый старт KeySeller Bot

## Шаг 1: Получите токен бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и сохраните полученный **токен**

## Шаг 2: Настройте .env файл

Откройте файл `.env` и заполните все поля:

```env
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
MANAGER_USERNAME=your_manager_username
CHANNEL_LINK=https://t.me/your_channel
PAYMENT_LINK=https://example.com/payment
```

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен от @BotFather |
| `MANAGER_USERNAME` | Username менеджера (без @) |
| `CHANNEL_LINK` | Ссылка на ваш Telegram канал |
| `PAYMENT_LINK` | Ссылка на страницу оплаты |

## Шаг 3: Запустите бота

```bash
cd /home/user/key-seller-bot
./start.sh
```

## 🐳 Запуск на сервере (systemd)

Создайте файл `/etc/systemd/system/keybot.service`:

```ini
[Unit]
Description=KeySeller Telegram Bot
After=network.target

[Service]
Type=simple
User=www-data
WorkingDirectory=/home/user/key-seller-bot
ExecStart=/home/user/key-seller-bot/venv/bin/python /home/user/key-seller-bot/bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

Активируйте:

```bash
sudo systemctl enable keybot
sudo systemctl start keybot
```
